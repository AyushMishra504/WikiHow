# 🧬 WikiDive

> A 3D knowledge graph explorer — start anywhere and fall into a rabbit hole of connected ideas.

---

## 🌌 Overview

WikiDive transforms the way you explore information.

Instead of navigating through pages, you move through **ideas**.

Start with any topic, and watch it expand into a living, interactive network — where each node represents a concept, and every connection reveals a new path to follow.

It’s not just search.
It’s exploration.

---

## ✨ Features

* 🌐 **3D Knowledge Graph**
  Visualize topics as dynamic, connected networks

* 🧠 **Rabbit Hole Exploration**
  Seamlessly dive deeper into related ideas

* ⚡ **Real-time Interaction**
  Smooth animations, hover effects, and immersive navigation

* 🎯 **Focus on Understanding**
  See how concepts relate instead of reading them in isolation

---

## 🖥️ Tech Stack

* **Frontend:** React
* **3D Rendering:** Three.js
* **Animations:** Custom shaders + GPU particles
* **Styling:** CSS (dark, minimal, cinematic UI)

---

## 🚀 Getting Started

```bash
# Clone the repo
git clone https://github.com/AyushMishra504/WikiHow.git

# Navigate into the project
cd WikiHow

# Install dependencies
npm install

# Run the app
npm run dev
```
